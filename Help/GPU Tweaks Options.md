# #1 Nvidia Tweaks
The Nvidia GPU Tweaks, contain over 250+ registry changes ranging from disabling DynamicPstate to disabling HDCP. The Nvidia Tweaks also include an Nvidia Profile Inspector Auto Nip Import, to change Nvidia Control Panel. By why stop there, the Nvidia GPU Tweaks within Oneclick contain a Custom Context Menu that allows the user to left-click the desktop and enable the Nvidia Container/Services with 1 click. The Nvidia GPU Tweaks also include a registry backup, meaning reverting just the GPU Tweaks is super easy!
